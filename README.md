# Claim Severity Modeling

## Overview

This project compares three statistical models for predicting insurance claim severity using the AutoClaims dataset.

The analysis compares:

- Ordinary Least Squares (OLS)
- Log-transformed Linear Regression
- Gamma Generalized Linear Model (Gamma GLM)

The objective is to identify the most appropriate model for predicting insurance claim severity.
